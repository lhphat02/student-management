
# SE107 - Quan Ly Hoc Sinh
A website for managing student's information



## Developers

## Tech Stack

**Client:** React.js, TailwindCSS

**Server:** Node, Next.js

**Database:** MongoDB
## Deployment

Before you try to deploy run

```bash
  npm install
```

To deploy this project run

```bash
  npm run dev
```

Then check it out at http://localhost:3000/
